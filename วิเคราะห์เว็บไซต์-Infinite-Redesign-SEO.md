## 1. บทสรุปผู้บริหาร (Executive Summary)

เว็บไซต์ `infinitematerialtech.com` เป็นเว็บขายสินค้ากลุ่ม **สีกันไฟ / วัสดุกันไฟ** (แบรนด์ Neocoat) ที่มี "ของดี" ครบ — สินค้าจริง ราคาชัดเจน มาตรฐานระดับสากล (ISO 834, ASTM E119) และมีวิศวกรโยธารับรอง ซึ่งเป็นจุดขายที่คู่แข่งส่วนใหญ่ไม่มี

แต่ปัจจุบันเว็บ **ยังใช้เทมเพลตสำเร็จรูปที่ดีไซน์เก่า** โครงสร้างหน้าไม่ได้ถูกออกแบบมาเพื่อ "ปิดการขาย" และมีปัญหาเชิงเทคนิค/SEO หลายจุดที่ทำให้เสียโอกาสทั้งด้านอันดับ Google และการเปลี่ยนผู้เข้าชมเป็นลูกค้า (conversion)

**ข้อสรุป 3 ข้อหลัก**

1. **ดีไซน์ & UX** — ไม่มี Hero / จุดขายชัดเจนหน้าแรก, ไม่มีปุ่ม "ขอใบเสนอราคา", พึ่ง LINE/โทรอย่างเดียว, ดีไซน์ดูไม่ทันสมัยและไม่รองรับมือถือเต็มที่
2. **SEO เชิงเทคนิค** — Title/Meta ไม่ตรงคีย์เวิร์ด, ไม่มี XML Sitemap, robots.txt บล็อกรูปสินค้า, ไม่มี Schema, H1 ผิดหลัก, URL รก และมี slug ผิด (หน้า "เกี่ยวกับเรา" ใช้ URL ว่า *ระบบกันซึม* / หน้า "ค่าบริการ" ใช้ URL ว่า *ฟื้นฟูผู้ป่วย* ซึ่งเป็นเศษเทมเพลตเก่า)
3. **คอนเทนต์ & ความน่าเชื่อถือ** — มีคำสะกดผิดหลายจุดในหน้าแรก (กระทบความน่าเชื่อถือของสินค้าด้านความปลอดภัย), ไม่มีบทความให้ความรู้เพื่อจับคีย์เวิร์ดกลุ่มค้นหาข้อมูล และไม่ได้ดึงจุดขายเรื่องมาตรฐาน/ใบรับรองขึ้นมาโชว์

การลงทุน redesign + วาง SEO อย่างเป็นระบบ จะช่วยให้เว็บ **ติดอันดับคีย์เวิร์ดหลัก "สีกันไฟ" และคีย์เวิร์ดยาว** ได้ดีขึ้น พร้อมเพิ่มอัตราการขอใบเสนอราคาจากผู้เข้าชมเดิม

## 4. จุดแข็งที่ต้องรักษาไว้ (Keep)

ก่อนจะรื้อ ต้องย้ำว่าเว็บนี้มี "ทุน" ที่ดีอยู่แล้ว และควรต่อยอด ไม่ใช่ทิ้ง:

- **แสดงราคาสินค้าจริงและราคาโปรโมชัน** อย่างโปร่งใส — เป็นข้อได้เปรียบที่คู่แข่งหลายรายไม่ทำ
- **มาตรฐานระดับสากล** (ISO 834, ASTM E119, FSRG 2019/035) และ **วิศวกรโยธารับรอง** — จุดขายที่ทรงพลังที่สุด
- **สินค้าครบไลน์** ตั้งแต่สีกันไฟ สีรองพื้น ทินเนอร์ ซีเมนต์กันไฟ ไปจนถึงผ้ากันไฟ
- **ช่องทางติดต่อหลากหลาย** (โทร 3 เบอร์, LINE, อีเมล, Facebook, แผนที่)
- **เป็นเจ้าของแบรนด์ Neocoat** — สร้าง brand keyword ได้

## 5. สรุปปัญหาแยกหมวด + ระดับความสำคัญ

ระดับ: **P0** = เร่งด่วนมาก (ทำก่อน) · **P1** = สำคัญ · **P2** = ควรทำ · **P3** = ทำเสริม

### 5.1 ดีไซน์ & UX
| ปัญหา | ระดับ |
|---|---|
| ไม่มี Hero / จุดขาย / CTA หลักบนหน้าแรก | P0 |
| ไม่มีฟอร์มขอใบเสนอราคา (พึ่ง LINE/โทร) | P0 |
| ดีไซน์เก่า ไม่รองรับมือถือเต็มที่ + viewport ล็อกซูม | P1 |
| ไม่มีการจัดลำดับสายตา (visual hierarchy) และช่องไฟ | P1 |
| ไม่โชว์ใบรับรอง/ผลงาน/รีวิว เพื่อสร้างความเชื่อมั่น | P1 |

### 5.2 คอนเทนต์ & ความน่าเชื่อถือ
| ปัญหา | ระดับ |
|---|---|
| คำสะกดผิดหลายจุด (วิศกร→วิศวกร, ทั่งนี้→ทั้งนี้, มาตรฐการ→มาตรฐานการ, สักันไฟ→สีกันไฟ) | P0 |
| ไม่มีบทความให้ความรู้ (จับคีย์เวิร์ดกลุ่มค้นหาข้อมูล) | P1 |
| ใช้อีเมล Gmail แทนอีเมลโดเมนบริษัท | P2 |
| ไม่มีหน้า FAQ / วิธีเลือกสีกันไฟ / คำนวณปริมาณ | P2 |

### 5.3 SEO เชิงเทคนิค
| ปัญหา | ระดับ |
|---|---|
| ไม่มี XML Sitemap (`sitemap.xml` ว่างเปล่า) | P0 |
| robots.txt บล็อกโฟลเดอร์รูปสินค้า `/user_file/` `/images/` | P0 |
| Title / Meta description ไม่ตรงคีย์เวิร์ด + สะกดผิด + ถูกตัด | P0 |
| ไม่มี Structured Data / Schema (Organization, Product, LocalBusiness, FAQ, Breadcrumb) | P1 |
| H1 ผิดหลัก (ใช้ "สินค้าขายดี") | P1 |
| og:url / og:image ใช้ `http://` (ควรเป็น `https://`) | P2 |
| ใช้ meta keywords (แท็กที่ Google เลิกใช้แล้ว) | P3 |
| URL ยาว รก และมี slug ผิดความหมาย (About=`/ระบบกันซึม-สีกันไฟ`, ค่าบริการ=`/ฟื้นฟูผู้ป่วย-สีกันไฟ`) | P2 |

## 6. ข้อเสนอการออกแบบใหม่ (Redesign) — หน้าแรก

ด้านล่างคือแนวทางออกแบบ **หน้าแรก** ใหม่ ในรูปแบบไกด์ไลน์เพื่อใช้พรีเซนต์ลูกค้า

**ภาพที่ 1 — ดีไซน์หน้าแรก (Homepage Design)**

![ดีไซน์หน้าแรกใหม่ของ Infinite Material & Technology](screenshots/03-homepage-design.png)

**ภาพที่ 2 — ไกด์ไลน์ดีไซน์ (สี · ตัวอักษร · ปุ่ม · องค์ประกอบ · หลักการ)**

![Design Guideline สำหรับหน้าแรก](screenshots/04-design-guideline.png)

### 6.1 หลักคิดของดีไซน์ใหม่
ปรับจาก "หน้าแคตตาล็อกสินค้า" → เป็น **"หน้าที่ปรึกษา + ปิดการขาย"** โดยดึงจุดแข็ง (มาตรฐาน + วิศวกรรับรอง + ราคาโรงงาน) ขึ้นมาเป็นหัวใจของหน้า

### 6.2 โครงสร้างหน้าแรกใหม่ (ตามภาพ)
1. **Header เหนียว (sticky)** — โลโก้ + เมนูจัดใหม่ + เบอร์โทร + ปุ่ม **"ขอใบเสนอราคา"** เด่นทุกหน้า
2. **Hero** — พาดหัวที่มีคีย์เวิร์ด ("สีกันไฟโครงสร้างเหล็ก ทนไฟ 1–3 ชม.") + จุดขาย + CTA คู่ (ขอใบเสนอราคา / ปรึกษา LINE) + ภาพงานจริง + ป้ายใบรับรอง (FSRG 2019/035)
3. **Trust Bar** — แถบมาตรฐาน ISO 834 · ASTM E119 · วิศวกรรับรอง · ราคาโรงงาน · ส่งทั่วประเทศ (เห็นทันทีไม่ต้องเลื่อน)
4. **หมวดสินค้าแบบการ์ด + ไอคอน** — 6 หมวดชัดเจน (ดีทั้ง UX และ internal link)
5. **สินค้าขายดี** — การ์ดใหม่ รูปคมชัด (เปิด index รูป) ราคาโปร่งใส + ปุ่ม CTA
6. **ทำไมต้องเลือกเรา** — 4 เหตุผล (ผ่านทดสอบจริง / วิศวกรรับรอง / ราคาโรงงาน / ส่งทั่วประเทศ)
7. **ขั้นตอนสั่งซื้อ–ปรึกษา 4 ขั้น** — ลดความลังเลก่อนตัดสินใจ
8. **แถบขอใบเสนอราคา (Quote band) + ฟอร์ม** — เก็บ Lead ออนไลน์ 24 ชม.
9. **Footer สะอาด** — อีเมลโดเมน + ลิงก์ Sitemap/Privacy + ช่องทางติดต่อครบ

### 6.3 ระบบดีไซน์ (Design System) ที่แนะนำ
- **โทนสี:** กรมท่าเข้ม (โครงสร้าง/ความน่าเชื่อถือ) + แดง-ส้มความปลอดภัย (accent/CTA) + เทาอ่อน (พื้นหลัง)
- **ฟอนต์:** ฟอนต์ไทยสมัยใหม่ อ่านง่าย (เช่น Sarabun / IBM Plex Sans Thai / Noto Sans Thai)
- **องค์ประกอบ:** การ์ดมุมมน, ไอคอน, ช่องไฟเยอะขึ้น, ปุ่ม CTA สีเดียวสม่ำเสมอทั้งเว็บ
- **มือถือเป็นหลัก (Mobile-first):** ปลดล็อก viewport, ปุ่มโทร/LINE ลอยล่างจอบนมือถือ, รูปโหลดแบบ lazy-load

### 6.4 หน้าใหม่ที่ควรมี
- หน้าหมวดสินค้าแยกชัด (สีกันไฟ / ซีเมนต์กันไฟ / ผ้ากันไฟ / ทินเนอร์ ฯลฯ) — ทำเป็น landing page จับคีย์เวิร์ด
- หน้ารายละเอียดสินค้าแบบมาตรฐาน (สเปก + ปริมาณการใช้ + ดาวน์โหลด TDS/ใบรับรอง + FAQ + ปุ่มขอราคา)
- **หน้า "บทความ/ความรู้"** (คลังคอนเทนต์ SEO)
- หน้า "ผลงาน/โครงการอ้างอิง" (case studies)
- หน้า "มาตรฐาน & ใบรับรอง"
- หน้า "ติดต่อ" ที่มีฟอร์ม + แผนที่ + ข้อมูลติดต่อครบ

## 7. แนวทางการทำ SEO

### 7.1 Technical SEO (ฐานราก — ทำก่อน)
- [ ] สร้าง **XML Sitemap** และส่งใน Google Search Console
- [ ] แก้ **robots.txt** ให้ **ปลดบล็อกรูปสินค้า** (`/user_file/`, `/images/`) เพื่อให้ Google เก็บรูป
- [ ] ตั้งค่า **Google Search Console + Bing Webmaster** และเชื่อม GA4
- [ ] บังคับ **HTTPS** ทั้งเว็บ + แก้ og:url/og:image เป็น `https://` + ตั้ง canonical ให้ถูก
- [ ] ปลดล็อก **viewport** (เอา `maximum-scale=1` ออก) เพื่อ mobile-friendly/accessibility
- [ ] ปรับ **Core Web Vitals** (บีบอัดรูป WebP, lazy-load, ลดสคริปต์ที่ไม่จำเป็น เช่น Histats หากซ้ำซ้อน) — *แนะนำวัดผลจริงด้วย PageSpeed Insights ก่อน–หลัง*
- [ ] จัดโครงสร้าง **URL** ให้สั้น สื่อความหมาย และแก้ slug ที่ผิด (About, ค่าบริการ)

### 7.2 On-page SEO
- [ ] เขียน **Title/Meta** ใหม่ทุกหน้าให้ขึ้นต้นด้วยคีย์เวิร์ด เช่น
  - หน้าแรก: `สีกันไฟ Neocoat มาตรฐาน ISO 834 | ราคาโรงงาน รับรองโดยวิศวกร`
- [ ] แก้ **H1** หน้าแรกเป็นคีย์เวิร์ดหลัก + จัดลำดับหัวข้อ H2/H3 ให้ถูก
- [ ] **แก้คำสะกดผิดทั้งเว็บ** (สำคัญต่อ E-E-A-T)
- [ ] ใส่ **ALT text** ให้รูปทุกภาพ (ช่วย Image SEO + accessibility)
- [ ] วาง **internal link** ระหว่างหมวด–สินค้า–บทความ

### 7.3 Structured Data / Schema (ช่วยทั้ง Google และ AI Search)
- [ ] `Organization` + `LocalBusiness` (ที่อยู่ เบอร์ เวลาทำการ)
- [ ] `Product` + `Offer` (ชื่อ ราคา สต็อก) ในหน้าสินค้า
- [ ] `BreadcrumbList`, `FAQPage`, `Article` (ในบทความ)

### 7.4 Content / Keyword Strategy
โครงสร้างคีย์เวิร์ดแบบ **Pillar–Cluster** โดยหน้าแรก/หมวดจับคีย์เวิร์ดหลัก และบทความจับคีย์เวิร์ดข้อมูล

**คีย์เวิร์ดกลุ่มซื้อ (Transactional):**
สีกันไฟ, สีทนไฟ, สีกันไฟราคา, ราคาสีกันไฟต่อตารางเมตร, สีกันไฟโครงสร้างเหล็ก, สีกันไฟ intumescent, ซีเมนต์กันไฟ, ผ้ากันไฟ, สีเซรามิคสะท้อนความร้อน, ทินเนอร์ 3A, รับพ่นสีกันไฟ, สีกันไฟ Neocoat

**คีย์เวิร์ดกลุ่มข้อมูล (Informational) — ทำเป็นบทความ:**
สีกันไฟคืออะไร, สีกันไฟทำงานอย่างไร, วิธีเลือกสีกันไฟ, สีกันไฟทนไฟกี่ชั่วโมง, มาตรฐาน ISO 834 / ASTM E119 คืออะไร, กฎกระทรวงเรื่องการกันไฟโครงสร้างเหล็ก, วิธีคำนวณปริมาณสีกันไฟ, ความต่างสีกันไฟสูตรน้ำ vs สูตรน้ำมัน

**ตัวอย่างบทความชุดแรก (10 หัวข้อ):**
1. สีกันไฟคืออะไร ทำงานอย่างไร แบบเข้าใจง่าย
2. สีกันไฟทนไฟได้กี่ชั่วโมง เลือกอย่างไรให้ผ่านกฎหมาย
3. สีกันไฟสูตรน้ำ vs สูตรน้ำมัน ต่างกันอย่างไร
4. วิธีคำนวณปริมาณสีกันไฟ/ราคาต่อตารางเมตร
5. มาตรฐาน ISO 834 และ ASTM E119 อธิบายง่ายๆ
6. ขั้นตอนการทา/พ่นสีกันไฟโครงสร้างเหล็กที่ถูกต้อง
7. ซีเมนต์กันไฟ vs สีกันไฟ เลือกแบบไหนดี
8. เช็กลิสต์ระบบกันไฟสำหรับโรงงาน/อาคาร
9. ผ้ากันไฟ Fiberglass ใช้งานอย่างไร ทนได้กี่องศา
10. รวมคำถามที่พบบ่อยเกี่ยวกับสีกันไฟ (FAQ)

### 7.5 Local SEO
- [ ] ตั้งค่า/ยืนยัน **Google Business Profile** (หมวดหมู่ ที่อยู่ เบอร์ รูป รีวิว)
- [ ] ให้ข้อมูล **NAP** (ชื่อ–ที่อยู่–เบอร์) ตรงกันทุกที่ + ใส่ LocalBusiness schema

### 7.6 Off-page / Authority
- [ ] ลงข้อมูลใน directory อุตสาหกรรม/ก่อสร้างไทย, Facebook Page ให้สม่ำเสมอ
- [ ] ขอรีวิว/อ้างอิงจากโครงการที่เคยทำ (case study + backlink)

## 10. ตัวชี้วัดความสำเร็จ (KPI)

- อันดับคีย์เวิร์ดหลัก ("สีกันไฟ", "สีกันไฟราคา", "สีกันไฟโครงสร้างเหล็ก") ใน Google
- ปริมาณ **Organic traffic** และจำนวนหน้าที่ถูก index (GSC)
- จำนวน **Lead / การขอใบเสนอราคา** ต่อเดือน (จากฟอร์ม + LINE)
- คะแนน **Core Web Vitals / PageSpeed** (มือถือ) ก่อน–หลัง
- อัตราการเปลี่ยนผู้เข้าชมเป็นลูกค้า (Conversion rate)
